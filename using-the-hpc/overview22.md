## System Overview

CCLA HPC is a 1080-core computing cluster available to ARM investigators and users for ARM data analysis and visualization. The cluster consists of a 30-node Cray cluster with a total of 7.68 GB DDR4 memory per core. Each dual socket node consist of two Intel Xeon E5-2697V4 processors \(18 cores per processor, 36 cores per node\). It provides 57.6 TB fast Solid State Drive \(SSD\) storage and 100 TB parallel Lustre filesystem storage. It uses a high bandwidth Mellanox InfiniBand network. The system has two \(2\) external login nodes. Two of the system nodes also provides NVIDIA Kepler3 K80 \(GK210\) GB GPUs.
